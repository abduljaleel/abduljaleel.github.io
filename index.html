AI Coding Agent

ax

A fast, provider-agnostic AI coding agent — built without the trade-offs.

A terminal-native coding agent with local-first architecture and security-first defaults. Own your agent instead of renting it from a single vendor.

  • Routes across Anthropic, OpenAI, Gemini, Ollama & vLLM by cost and quality
  • Sub-millisecond trigram code search — 100–1000× faster than grep
  • Built-in secret & prompt-injection scanning
  • 30+ tools, persistent memory, 80% test coverage

MCP Server

Wharf

A dock for your Managed Agents.

An MCP server that exposes Anthropic's Managed Agents API through a conversational interface. Deploy, message, update and retire autonomous agents — all from Claude, without leaving the chat.

  • Seven MCP tools, from deploy_agent to delete_agent
  • Single-file Cloudflare Worker — stateless by design
  • Fire-and-forget polling for long-running tasks
  • Optional shared-secret authentication

Supply-Chain Security

VIM

Finding bugs is cheap. Tracking where they live is not.

Vulnerability Inheritance Map — an open-source graph that traces how vulnerabilities propagate across forks, vendored dependencies and compiled binaries. The lineage layer today's SCA tools miss.

  • Trilateral identity: pURL, SWHID & TLSH
  • Interoperates with GUAC, OSV, deps.dev & Sigstore
  • Finds vulnerable copies outside the dependency tree
  • Evidence built for EU Cyber Resilience Act deadlines

Cloud Dev Tool

LoopbackCloud

The whole cloud, on localhost.

A local-first emulator that runs AWS, Azure and GCP on your own machine. Point your existing SDKs, CLIs and IaC at localhost and build multicloud apps with no accounts, no spend and no network.

  • 70+ services across AWS, Azure & GCP in one local stack
  • Drop-in for standard SDKs, CLIs, Terraform & CDK
  • Unified console for resources across all three clouds
  • Native GraalVM image — 24 ms cold start, MIT licensed
